# 项目范围

## 目标

MyInvestTAA 的目标是形成可重复、可解释、可审核的战术资产配置决策支持系统。系统研究资产比例，展示数据、模型、映射和执行限制，并通过只读报告帮助用户人工判断。

核心能力包括：

- 读取在信号日期当时可获得的数据。
- 把研究信号安排到下一个有效交易日。
- 通过已批准 mapping 将研究资产转换为 ETF 或明确的现金原因桶。
- 对上市前、无价格、无代理和低质量代理作出确定处理。
- 对已持有缺价 ETF 保留最近有效估值。
- 对无法完成的持仓调整建立、重试、完成或 supersede pending。
- 对 requested、actual、cash 和 deferred 状态进行对账。
- 在相同输入下产生确定输出。
- 始终保持非交易指令边界。

## 核心固定假设

- 交易成本为零。
- 滑点为零。
- 现金收益为零。
- 不借款。
- 不使用 ETF 上市前收益。
- 不用指数收益替代 ETF 收益。

这些是假设边界，不是待扩展配置。

## 当前阶段

项目处于 Execution Core Stabilization 阶段：

1. C0-A 冻结范围、合同和版本生命周期，不改变运行结果。
2. C0-B 抽取无 V1 依赖的核心接口，保持 B1 业务逐点等价。
3. C0-C 清理兼容入口并建立架构护栏。
4. R0 锁定依赖和 CI，只做工程可靠性，不增加业务功能。
5. 最后单独执行 V1/V2 迁移裁决。

完成结构整理不代表 V2 自动成为正式执行核心。

## 输出边界

所有配置和执行结果使用比例、状态、原因和指标表达。系统不输出订单、股数、金额、目标价格或自动交易动作。
